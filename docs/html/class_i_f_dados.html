◆ run()

virtual void IFDados::run ( Conta )
pure virtual

Inicia a IFDados.

Inicia a IFDados. O usuário deve possuir a opção de visualizar seu perfil ou acessar as Viagens registradas em sua Conta. Também oferece a opção de desautenticar a Conta (retornar void).
Se, durante a execução do perfil, o retorno for verdadeiro, deve-se imediatamente desautenticar a Conta (retornar void).

Parâmetros
Contaatualmente autenticada.
