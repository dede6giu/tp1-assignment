bool SBAutenticacao::autenticar ( Conta )
virtual

Autentica o usuário.

O método recebe um Código e uma Senha, ou seja, uma Conta. Ele checa se a Senha é a associada ao Código fornecido no banco de dados e, se sim, retorna true. Se o código não existe, ou a senha está incorreta, retorna false.

Parâmetros
Contaa ser autenticada.
Retorna
Sucesso da operação.
