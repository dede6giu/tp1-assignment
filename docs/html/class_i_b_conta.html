Interface Back Conta.

232013031 - Implementação e Documentação
A IBConta é responsável pela comunicação do usuário com o banco de dados quanto ao controle de Conta. Ela permite criar, excluir, ler e atualizar uma Conta.

virtual bool IBConta::atualizar ( Conta ,
Senha  )
pure virtual

Atualiza a Senha de uma Conta.

Recebe uma Conta e uma Senha. Após verificar a existência da Conta (e se a Senha fornecida com ela está correta), altera-se o valor da Senha para o novo valor fornecido. Senão, a operação falha.

Parâmetros
Contaque trocará de Senha.
NovaSenha.
Retorna
Sucesso da operação.
